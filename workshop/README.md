# AI Bootcamp: Build a Full-Stack App with Claude Code

A fully automated, AI-driven bootcamp where QA engineers learn Claude Code by building a professional React web app from scratch — without writing a single line of code manually.

## What You Will Build

**QA Command Center** — a test management dashboard with:

- Dashboard with metrics cards and trend charts
- Test case CRUD with ISTQB severity levels
- Bug tracker with GitHub issue sync
- Test suite management and test run executor
- Jira Cloud integration (epics, stories, acceptance criteria)
- HTML/PDF test report export
- CI/CD pipeline with automated code review
- Settings and CSV import/export

**Tech stack**: Express + React (Vite) + SQLite — all generated by Claude Code.

## Prerequisites

### Required

- Claude Code CLI installed and authenticated (`claude --version`)
- Node.js 18+ and npm (`node --version`)
- Git configured (`git config --global user.name`)
- GitHub account with a personal access token
- Python 3.10+ (for bootcamp platform tools)

### For Jira Integration (Sessions 5+)

- Jira Cloud account (free tier works)
- Atlassian API token

### Recommended

- VS Code or JetBrains IDE with Claude Code extension
- Basic familiarity with ChatGPT or similar AI tools

## Getting Started

### 1. Student Setup

```bash
# Clone the repository
git clone <repo-url>
cd claude-howto

# Run the bootcamp setup
cd workshop
./bootcamp setup
```

The setup wizard will:

- Ask for your student ID and name
- Create your local bootcamp directory (`~/.claude-bootcamp/`)
- Install prompt logging hooks into your Claude Code configuration
- Link you to the active cohort

### 2. Starting a Session

```bash
# Begin session 1
./bootcamp start-session 1
```

This validates that all prerequisites are met and unlocks the session guide.

### 3. Working Through a Session

1. Read the session guide in `sessions/s01-slash-commands/README.md`
2. Follow the goals and requirements — craft your own prompts (no copy-paste)
3. Check the deliverables list to know when you are done
4. Self-verify using the verification checklist

### 4. Completing a Session

```bash
# Submit session 1 for validation
./bootcamp complete-session 1
```

This runs automated gate checks, records your score, and unlocks the next session.

### 5. Checking Progress

```bash
./bootcamp status
```

## Scoring

Your work is automatically scored on a 0-100 scale. See [SCORING-RUBRIC.md](SCORING-RUBRIC.md) for full details.

| Component | Weight |
|-----------|--------|
| Prompt quality | 25% |
| Efficiency | 15% |
| Deliverable quality | 20% |
| Standards compliance | 15% |
| Final task (Session 10) | 25% |

## Falling Behind?

If you miss a session or need to catch up, you can restore to any session checkpoint:

```bash
./bootcamp start-session N  # Automatically offers catch-up if previous sessions incomplete
```
